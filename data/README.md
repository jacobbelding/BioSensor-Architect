# Data

## `example_constructs/`
Pre-built example outputs showing what a complete design workflow produces.

- `nitrogen_reporter.json` — Nitrate reporter construct based on the NRT2.1 promoter and betanin visible reporter system.

## `literature_index/`
ChromaDB vector store for the RAG-indexed literature collection. This directory is gitignored — run `bsa index-papers` to populate it.
